## Project Overview

> MoodLog is a web-based mood tracking application developed using React.
> It allows users to record their daily emotions such as happy, sad, angry, or neutral.
> The system stores mood history and helps users analyze emotional patterns over time.
> The main goal of this project is to promote mental awareness and help users better understand their emotions.

---
